# 🖥️ Sistemas Informáticos - Proyecto Intermodular

## 📌 Descripción

Este apartado del proyecto corresponde al módulo de **Sistemas Informáticos**, donde se ha definido, configurado y documentado el entorno necesario para la ejecución de la aplicación empresarial **Distribuciones Barquero**.

El objetivo principal ha sido simular un entorno real de trabajo, asegurando que la aplicación funcione correctamente desde el punto de vista técnico, incluyendo instalación, configuración, ejecución y mantenimiento del sistema.

---

## ⚙️ Contenido del proyecto

En esta carpeta se incluye:

* 📄 **Informe técnico (PDF)**
  Documento completo donde se describe:

  * Tipo de sistema utilizado
  * Requisitos hardware y software
  * Sistema operativo recomendado
  * Proceso de instalación
  * Usuarios y permisos
  * Mantenimiento del sistema
  * Despliegue y pruebas

* 📁 **Capturas de pantalla (/img)**
  Evidencias visuales del funcionamiento del sistema:

  * Base de datos en MySQL
  * Aplicación Java en ejecución
  * Estructura del proyecto
  
---

## 💻 Tecnologías utilizadas

* **Java + JavaFX** → Aplicación de escritorio
* **MySQL** → Base de datos
* **JDBC** → Conexión entre aplicación y base de datos
* **HTML, CSS, JavaScript** → Web corporativa
* **IntelliJ IDEA / VS Code** → Entornos de desarrollo
* **GitHub** → Control de versiones

---

## 🎯 Objetivo del documento

El informe técnico tiene como finalidad demostrar que:

* El sistema puede ejecutarse en un entorno definido
* La aplicación arranca correctamente
* Todas las funcionalidades operan de forma estable
* El entorno es reproducible en otros equipos

Se ha conseguido un entorno funcional donde:

* La aplicación se ejecuta sin errores
* La base de datos responde correctamente
* El sistema permite gestionar clientes, productos y pedidos
