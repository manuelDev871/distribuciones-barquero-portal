# 🚀 Proyecto: Distribuciones Barquero Portal

## 📌 Descripción
Este repositorio contiene el desarrollo del proyecto "Distribuciones Barquero Portal".  
Se ha subido el proyecto completo junto con un video explicativo donde se muestra su funcionamiento y estructura.

El objetivo es construir una aplicación organizada, aplicando buenas prácticas de programación y uso de herramientas como Git y GitHub.

## 📂 Estructura del proyecto
El código está organizado en diferentes carpetas para facilitar su mantenimiento:

- `src/` → Código fuente principal  
- `main/` → Archivos principales del proyecto  
- `resources/` → Configuración

*(La estructura puede ampliarse en futuras versiones del proyecto)*
