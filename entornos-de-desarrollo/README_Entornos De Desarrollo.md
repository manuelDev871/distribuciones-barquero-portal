# Portal Web Corporativo y Sistema de Gestión

## Distribuciones Barquero

Proyecto intermodular del ciclo formativo **Técnico Superior en
Desarrollo de Aplicaciones Web (DAW)**.

------------------------------------------------------------------------

# 📌 Descripción del proyecto

Este proyecto consiste en el desarrollo de un portal web corporativo
junto con una aplicación de gestión empresarial para la empresa
Distribuciones Barquero.

El objetivo es modernizar su presencia digital y permitir la gestión de
clientes, productos y pedidos mediante una aplicación conectada a base
de datos.

------------------------------------------------------------------------

# 🎯 Objetivos del proyecto

-   Diseñar una web corporativa moderna
-   Crear un sistema de gestión empresarial
-   Implementar base de datos relacional
-   Aplicar buenas prácticas de programación
-   Utilizar control de versiones con Git

------------------------------------------------------------------------

# ▶️ Ejecución del proyecto

Para ejecutar la aplicación:

1.  Instalar Java (JDK 17 o superior)
2.  Instalar MySQL

Pasos:

1.  Clonar el repositorio: git clone
    https://github.com/tu-usuario/distribuciones-barquero-portal

2.  Importar el proyecto en IntelliJ IDEA

3.  Crear la base de datos en MySQL

4.  Importar los scripts SQL incluidos en el proyecto

5.  Configurar la conexión en la clase Conexion.java

6.  Ejecutar la clase MainApp
