# Portal Web Corporativo y Sistema de Gestión

## Distribuciones Barquero

Proyecto intermodular del ciclo formativo **Técnico Superior en Desarrollo de Aplicaciones Web (DAW)**.

---

# 📌 Descripción del proyecto

Este proyecto consiste en el **diseño y desarrollo de un portal web corporativo moderno junto con una aplicación de gestión empresarial** para la empresa **Distribuciones Barquero**, una empresa dedicada a la distribución de productos alimentarios, bebidas y snacks a establecimientos de hostelería y comercios.

Actualmente la empresa cuenta con páginas web antiguas y con funcionalidades limitadas. El objetivo de este proyecto es proponer una **modernización de su presencia digital**, desarrollando una plataforma que permita presentar la empresa de forma profesional y gestionar parte de su actividad comercial.

---

# 🎯 Objetivos del proyecto

Los principales objetivos de este proyecto son:

* Diseñar una **web corporativa moderna y accesible**
* Crear un **catálogo de productos digital**
* Simular un **sistema de pedidos para clientes**
* Desarrollar una **aplicación de gestión empresarial**
* Implementar una **base de datos estructurada**
* Aplicar **buenas prácticas de programación**
* Utilizar **Git y GitHub para el control de versiones**
* Documentar el proyecto como si fuera un entorno profesional

# 🌐 Funcionalidades del portal web

El portal web corporativo incluirá las siguientes secciones:

* Página de inicio
* Información sobre la empresa
* Catálogo de productos
* Información de servicios
* Página de contacto
* Zona de clientes

La web estará diseñada con un **estilo moderno y responsive**, adaptándose a diferentes dispositivos.

---

# 🗄 Sistema de gestión empresarial

El proyecto incluye una aplicación desarrollada en Java que permitirá gestionar información de la empresa.

Las funcionalidades principales incluyen:

* Gestión de productos
* Gestión de clientes
* Creación de pedidos
* Consulta de pedidos
* Gestión básica de stock

La aplicación se conectará a la base de datos mediante **JDBC**, permitiendo realizar operaciones de tipo **CRUD (Create, Read, Update, Delete)**.

---

# 🗄 Modelo de datos

La base de datos del proyecto representará el funcionamiento de una empresa de distribución.

Entre las entidades principales se encuentran:

* Clientes
* Productos
* Categorías
* Pedidos
* Detalle de pedidos
* Proveedores
* Empleados

El diseño se realizará mediante un **diagrama entidad-relación**, que posteriormente se transformará en un **modelo relacional implementado en SQL**.
