## 1. Análisis de datos

El proyecto representa una empresa llamada **Distribuciones Barquero**, dedicada a la gestión y distribución de productos a clientes.

### Tipo de empresa

Empresa de distribución que gestiona pedidos, productos, clientes y proveedores.

### Información que maneja

La empresa necesita almacenar y gestionar:

* Datos de clientes
* Información de empleados
* Catálogo de productos
* Clasificación de productos por categorías
* Proveedores de productos
* Pedidos realizados por clientes
* Detalles de cada pedido (productos incluidos, cantidad, precio)

### Entidades identificadas

A partir del análisis del negocio se identifican las siguientes entidades:

* clientes
* pedidos
* detallePedido
* productos
* empleado
* categorias
* proveedor

### Objetivo del sistema

El objetivo de la base de datos es representar el funcionamiento interno de la empresa, permitiendo gestionar pedidos y relacionarlos con clientes, empleados, productos, categorías y proveedores.
